# react-vite-template

这是一个从生产项目中提炼出来的模板

## 特性

- [x] Ant Design 5 + TailwindCSS
- [ ] 主题切换
- [x] 严格的lint和提交检测
- [ ] Docker
- [x] 组件测试
- [x] e2e测试
- [ ] CI
